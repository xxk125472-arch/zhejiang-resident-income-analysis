浙江省2022—2024年城乡居民人均可支配收入分析项目

一、项目简介

本项目基于浙江省统计局官网发布的《浙江统计年鉴》相关数据，选取2022—2024年浙江省全体居民、城镇常住居民和农村常住居民的人均可支配收入及生活消费支出数据，使用 Excel 和 MySQL 完成数据整理、指标计算、趋势分析、城乡差距分析、收入结构分析和收支关系分析。

项目主要围绕“浙江省2022—2024年城乡居民人均可支配收入变化情况”展开，重点分析城镇居民与农村居民在收入水平、收入增速、城乡收入差距、城乡收入倍差以及消费倾向方面的差异。

通过本项目，可以完整展示从官方数据获取、Excel 数据清洗与可视化、MySQL 数据建模与查询分析，到最终项目总结与结果展示的完整数据分析流程。


二、项目背景

居民人均可支配收入是衡量居民生活水平和经济发展质量的重要指标。通过分析城镇居民和农村居民人均可支配收入的变化，可以了解城乡居民收入增长情况，也可以进一步观察城乡收入差距是否扩大或缩小。

浙江省作为经济较发达地区，居民收入水平整体较高，但城镇和农村居民之间仍然存在一定差距。本项目通过2022—2024年数据，对浙江省城乡居民收入变化进行分析，尝试回答以下问题：

1. 2022—2024年浙江省居民人均可支配收入是否持续增长？
2. 城镇居民和农村居民收入水平差距有多大？
3. 城乡收入绝对差距是否扩大？
4. 城乡收入倍差是否下降？
5. 农村居民收入增速是否高于城镇居民？
6. 居民收入结构中，工资性收入、经营净收入、财产净收入和转移净收入分别占比多少？
7. 城镇和农村居民的平均消费倾向是否存在差异？


三、数据来源

本项目使用的数据来自浙江省统计局官网发布的统计年鉴数据，主要包括以下三个 Excel 数据源：

1. 5-5 城乡居民人均可支配收入情况（2022-2024年）.xlsx
   用途：作为本项目收入分析的核心数据源，包含全体居民、城镇常住居民、农村常住居民在2022—2024年的人均可支配收入及收入来源分项数据。

2. 5-9 城乡居民人均收支情况（2022-2024年）.xlsx
   用途：用于分析居民可支配收入、生活消费支出、收支结余和平均消费倾向。

3. 5-3 城乡居民家庭人均生活消费支出（1978-2024年）.xlsx
   用途：作为消费支出长期数据来源，本项目主要截取其中2022—2024年的生活消费支出数据，与表5-9进行辅助验证和补充分析。

九、主要分析结论

1. 浙江省居民收入持续增长

2022—2024年，浙江省全体居民、城镇常住居民和农村常住居民人均可支配收入均保持增长。

全体居民人均可支配收入由2022年的60302元增长至2024年的67013元。
城镇居民人均可支配收入由2022年的71268元增长至2024年的78251元。
农村居民人均可支配收入由2022年的37565元增长至2024年的42786元。

说明浙江省居民收入水平整体呈现稳步提升趋势。

2. 城镇居民收入水平仍然明显高于农村居民

2024年，浙江省城镇居民人均可支配收入为78251元，农村居民人均可支配收入为42786元，城乡收入差距为35465元。

从收入水平看，城镇居民收入仍然显著高于农村居民。

3. 城乡收入绝对差距有所扩大

2022年城乡收入差距为33703元，2023年为34686元，2024年为35465元。

从绝对金额看，城乡居民收入差距有所扩大。

4. 城乡收入倍差持续下降

虽然城乡收入绝对差距扩大，但城乡收入倍差由2022年的1.90下降到2024年的1.83。

这说明城镇居民收入虽然仍高于农村居民，但农村居民收入增长速度相对更快，城乡收入相对差距有所缩小。

5. 农村居民收入增速高于城镇居民

2023年，城镇居民收入同比增长约5.23%，农村居民收入同比增长约7.31%。
2024年，城镇居民收入同比增长约4.34%，农村居民收入同比增长约6.14%。

农村居民收入增速连续高于城镇居民，是城乡收入倍差下降的重要原因。

6. 工资性收入是居民收入的主要来源

从收入结构看，工资性收入在全体居民可支配收入中占比最高，是居民收入的主要来源。

2024年，全体居民工资性收入为37220元，占可支配收入的比例约为55.54%。

7. 农村居民平均消费倾向高于城镇居民

2024年，城镇居民平均消费倾向约为64.80%，农村居民平均消费倾向约为77.23%。

这说明农村居民生活消费支出占收入的比例更高，收支结余空间相对较小。

十一、项目不足与改进方向

1. 时间跨度较短
   当前核心数据主要为2022—2024年，属于近三年分析。如果后续补充2020年和2021年数据，可以扩展为真正的近五年分析。

2. 分析维度仍可扩展
   当前主要分析收入和消费支出，后续可以进一步加入地区生产总值、人均GDP、城镇化率、就业数据等指标，分析收入增长背后的影响因素。

3. 可视化工具可以升级
   当前主要使用 Excel Dashboard，后续可以使用 Power BI、Tableau 或 Python 进行更高级的数据可视化。

4. 数据自动化程度可以提高
   当前数据主要手动整理，后续可以尝试使用 Python 自动读取 Excel、清洗数据并导入 MySQL，提高项目自动化程度。


十二、总结

本项目围绕浙江省2022—2024年城乡居民人均可支配收入展开，使用官方统计数据，结合 Excel 和 MySQL 完成了较完整的数据分析流程。

分析结果显示，浙江省居民收入整体持续增长，城镇居民收入仍明显高于农村居民，但农村居民收入增速较快，城乡收入倍差持续下降，说明城乡收入相对差距有所缩小。同时，农村居民平均消费倾向高于城镇居民，说明农村居民消费支出占收入比例更高，收支结余空间相对较小。

整体来看，本项目能够较好体现数据获取、数据清洗、指标计算、SQL建模、可视化展示和分析总结能力，适合作为数据分析入门作品集项目。
